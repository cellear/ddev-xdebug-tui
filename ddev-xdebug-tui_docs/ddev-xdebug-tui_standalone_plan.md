
# ddev-xdebug-tui_standalone_plan.md

## 1. Project Overview
Project Name: ddev-xdebug-tui
Type: DDEV add-on
Language: Go
UI Framework: tview
Scope: Minimal Xdebug terminal debugger for DDEV projects

ddev-xdebug-tui is a lightweight, terminal-native debugger for PHP projects running inside DDEV.
It allows CLI-first Drupal and Backdrop developers to debug code without integrating Xdebug into a full IDE.

It is intentionally minimal.
